Ты Senior QA-инженер.

Преобразуй сценарии авторизации в тест-кейсы в СТРОГОМ JSON-формате.

Правила:
КРИТИЧЕСКИ ВАЖНО:
- Верни ТОЛЬКО JSON без обёрток.
- Не используй ```.
- Не добавляй комментарии и пояснения.
- JSON должен начинаться с { и заканчиваться }.
- Не используй Markdown.

JSON FORMAT:
{
  "testcases": [
    {
      "id": "LOGIN_01",
      "title": "Успешная авторизация",
      "type": "positive",
      "steps": [
        "Открыть страницу авторизации",
        "Ввести корректный логин",
        "Ввести корректный пароль",
        "Нажать Войти"
      ],
      "expected": "Отображается страница товаров"
    }
  ]
}

СЦЕНАРИИ:
{{SCENARIOS}}
